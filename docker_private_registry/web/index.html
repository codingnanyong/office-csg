Setup Scripts

Automated installation and configuration scripts for Linux, macOS, and Windows.

Install Docker:
Linux/macOS: curl http://{REGISTRY_HOST}:9000/scripts/install-docker.sh -o install-docker.sh
Windows: curl http://{REGISTRY_HOST}:9000/scripts/install-docker-windows.bat -o install-docker-windows.bat
Setup Registry:
Linux/macOS: curl http://{REGISTRY_HOST}:9000/scripts/setup-docker-registry.sh -o setup.sh
Windows: curl http://{REGISTRY_HOST}:9000/scripts/setup-docker-registry-windows.bat -o setup.bat
Browse Scripts Directory
